AGENT RUNTIME TOOLS
SKILLS + TOOLS + DECISION LAYERS
Skills, MCP tools, model routing, and decision systems that make agents more capable.
BUILD THIS
Tools that improve how agents choose models, use MCPs, invoke skills, or reason about decisions.
NOT THIS
Do not use this road for full business workflows unless the main contribution is the runtime primitive itself.
SOFTWARE FACTORY
PIPELINES + FLEETS + CONTINUOUS IMPROVEMENT
Systems that continuously build, test, coordinate, and improve codebases.
BUILD THIS
Multi-step systems where several agents or stages continuously plan, build, verify, and improve software.
NOT THIS
Do not use this road for a single automation or single verifier. This road is about pipelines and coordination.
